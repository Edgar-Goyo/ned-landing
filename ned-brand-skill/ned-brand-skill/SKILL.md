# NED — Sistema de Identidad de Marca

NED es una empresa de automatización, desarrollo web, soluciones digitales y aplicaciones para negocios. Su identidad visual es oscura, tecnológica y profesional, inspirada en la arquitectura isométrica del logo: un cubo 3D dentro de una sala hexagonal.

Lee `references/colores.md` para la paleta completa y `references/voz.md` para el tono de comunicación.

## Logo

El logo oficial de NED es un cubo isométrico 3D dentro de una sala hexagonal:
- **Cubo central**: gris oscuro (`#3A3A3A`) con las letras N (cara superior), E (cara izquierda), D (cara derecha) grabadas en rojo oscuro/vino
- **Sala exterior**: hexágono en gris oscuro con piso y acentos en fucsia/vino
- **Archivo**: `logo_NED.jpg` o `logo_NED_transparent.png` (con fondo transparente — usar siempre este)
- **Nunca** recrear el logo con SVG ni con texto — siempre usar la imagen original
- En fondos oscuros: usar versión con fondo transparente
- Tamaño mínimo: 32px en interfaces, 1.5cm en impresos

## Tipografía

| Rol | Fuente | Peso | Uso |
|-----|--------|------|-----|
| Títulos display | Playfair Display | 700–900 | Heroes, títulos de sección, números grandes |
| Etiquetas técnicas | DM Mono | 300–500 | Tags, código, fechas, métricas, nav links |
| Cuerpo | DM Sans | 300–500 | Párrafos, descripciones, listas |

**Reglas tipográficas:**
- Títulos: Playfair Display, tamaño mínimo 28px en web / 24pt en documentos
- Etiquetas y monoespaciado: DM Mono con `letter-spacing: 0.08–0.15em` y `text-transform: uppercase`
- Cuerpo: DM Sans `font-weight: 300` para tono sofisticado
- Nunca usar Arial, Roboto, Inter ni fuentes del sistema como elección principal
- En presentaciones (.pptx): Georgia para títulos, Calibri para cuerpo (fuentes seguras de Office)
- Jerarquía: tamaño, peso y familia DEBEN variar entre niveles — nunca dos niveles iguales

## Estética visual

NED sigue una dirección **industrial-tech oscura con geometría isométrica**:

- **Tema**: siempre oscuro como base (`#1A1A1C`)
- **Grillas**: líneas sutiles de cuadrícula en el fondo aportan profundidad tecnológica
- **Geometría**: formas rectangulares con bordes de `2–4px radius` — nunca bordes muy redondeados
- **Efectos**: sombras con color `#7D1A3A33` (vino semitransparente), no sombras grises genéricas
- **Animaciones**: lentas y suaves (0.2–0.5s ease) — nunca rebotes ni efectos llamativos
- **Iconografía**: Tabler Icons outline, siempre en Fucsia NED o Vino NED
- **Imágenes**: si se usan fotos, deben tener un overlay oscuro con `opacity: 0.4–0.6`
- **Fondos de sección**: alternar entre `#1A1A1C` y `#2C2C2F` — nunca dos secciones del mismo color seguidas

**Para material impreso o PDF:**
- Fondo blanco `#FFFFFF` con acento en Vino NED `#7D1A3A`
- Tipografía en Grafito profundo `#1A1A1C`
- Logo siempre presente en esquina superior izquierda o centrado en portada
